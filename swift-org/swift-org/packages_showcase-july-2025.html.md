# Community Showcase: July 2025

 
 
 
 

 
 [View this month’s Community Showcase](/packages/showcase.html)
 

 
 The Community Showcase of Swift packages features packages from around the Swift community nominated via
 [this thread on the Swift forums](https://forums.swift.org/t/68168) and voted on by the
 [Swift Website Workgroup](https://swift.org/website-workgroup/).
 
 **This list is an archive of the Community Showcase from July 2025**.

  
- 
 
 
 swift-snapshot-testing-macros
 by Adam Carter
 
 `SnapshotTestingMacros` provides a macro-based approach for generating snapshot tests, using annotations to mark functions for Swift views. It supports customization through traits like device sizes and themes.

 Yams
 by JP Simard
 
 Yams is a Swift-based YAML parser utilizing LibYAML. It supports conversion between YAML and `Codable`, Swift Standard Library types, and Yams-native Node representations, minimizing computational overhead.

 Swift-TTS
 by Prince Canuma
 
 Text-to-speech and speech-to-speech library utilizing Apple’s MLX framework for efficient speech synthesis. Features include multi-language support, voice customization, adjustable speed, web interface, and REST API.
