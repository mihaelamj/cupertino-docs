# Community Showcase: March 2025

 
 
 
 

 
 [View this month’s Community Showcase](/packages/showcase.html)
 

 
 The Community Showcase of Swift packages features packages from around the Swift community nominated via
 [this thread on the Swift forums](https://forums.swift.org/t/68168) and voted on by the
 [Swift Website Workgroup](https://swift.org/website-workgroup/).
 
 **This list is an archive of the Community Showcase from March 2025**.

  
- 
 
 
 FileMonitor
 by aus der Technik - Simon & Simon GbR
 
 FileMonitor provides a unified API to detect file changes in a directory on Linux and macOS. It detects file creations, modifications, and deletions and then propagates events through a delegate function.

 Versionator
 by Elegant Chaos
 
 A plugin to gathers version information and embeds it into a package/executable, allowing retrieval at runtime. Generates `Version.swift`, `Info.plist`, and a header file.

 Numerix
