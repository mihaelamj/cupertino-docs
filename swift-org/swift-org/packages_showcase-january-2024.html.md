# Community Showcase: January 2024

 
 
 
 

 
 [View this month’s Community Showcase](/packages/showcase.html)
 

 
 The Community Showcase of Swift packages features packages from around the Swift community nominated via
 [this thread on the Swift forums](https://forums.swift.org/t/68168) and voted on by the
 [Swift Website Workgroup](https://swift.org/website-workgroup/).
 
 **This list is an archive of the Community Showcase from January 2024**.

  
- 
 
 
 soto
 by Soto for AWS
 
 Soto for AWS is a Swift language SDK for Amazon Web Services (AWS). It provides access to all AWS services via a direct mapping of the REST APIs Amazon publishes for each of its services.

 6.0+

 Nominated via [this forum post](https://forums.swift.org/t/nominations-for-the-packages-community-showcase-on-swift-org/68168/13).
