# Shared Directives

API Collection# Shared Directives

Directives supported by multiple directives in interactive tutorials.## [Topics](/documentation/docc/tutorial-shared-directives#topics)

### [Content Directives](/documentation/docc/tutorial-shared-directives#Content-Directives)

[`@Chapter(name: String)`](/documentation/docc/chapter)Organizes related tutorial pages into a chapter on a table of contents page.[`@Intro(title: String)`](/documentation/docc/intro)Displays an introduction section at the top of a table of contents or tutorial page.### [Media Directives](/documentation/docc/tutorial-shared-directives#Media-Directives)

[`@Image(source: ResourceReference, alt: String)`](/documentation/docc/image)Displays an image in a tutorial.[`@Video(source: ResourceReference, alt: String, poster: ResourceReference)`](/documentation/docc/video)Displays a video in a tutorial.
