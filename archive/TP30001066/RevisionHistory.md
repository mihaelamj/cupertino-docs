2010-11-19Updated for OS X v10.6 and iOS 4.2.

2010-06-25Minor clarifications and editing.

2009-05-18Updated the font names in text examples to reflect fonts available on both iOS and OS X.

2008-06-04Updated for iOS SDK.

 Added information about image formats to  [Bitmap Image Information](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-BBCJBICF).

 Corrected typos.

2007-12-11Revised text chapter and added a glossary.

 Added code that creates a dictionary and adds metadata to it. See [Listing 13-4](dq_pdf/dq_pdf.html#//apple_ref/doc/uid/TP30001066-CH214-CJBIGHBH).

2007-07-02Updated for OS X v10.5.

 Renamed the Shadings chapter to Gradients and revised it to include information on the use of the `CGGradientRef` opaque data type.

 In [Data Management in Quartz 2D](dq_data_mgr/dq_data_mgr.html#//apple_ref/doc/uid/TP30001066-CH216-TPXREF101) and a link and information about *[Image I/O Programming Guide](../ImageIOGuide/imageio_intro/ikpg_intro.html#//apple_ref/doc/uid/TP40005462)*. 

 Updated the introduction with recent, relevant related documentation and added a description of the revised [Gradients](dq_shadings/dq_shadings.html#//apple_ref/doc/uid/TP30001066-CH207-TPXREF101) chapter.

 Revised [Quartz 2D Opaque Data Types](dq_overview/dq_overview.html#//apple_ref/doc/uid/TP30001066-CH202-BBCGCGBA) to include `CGGradientRef` and provided links to information on `CGImageSourceRef` and `CGImageDestinationRef` opaque data types which are part of the Image I/O framework. 

 Updated [Table 2-1](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-CJBEAGHH) with additional pixel formats.

2007-01-08Fixed a number of minor technical issues.

 Improved the wording in the first paragraph of [Gradients](dq_shadings/dq_shadings.html#//apple_ref/doc/uid/TP30001066-CH207-TPXREF101).

 Made a correction to the floating-point gray information in [Table 2-1](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-CJBEAGHH).

 Corrected the declarations in [Listing 14-5](dq_pdf_scan/dq_pdf_scan.html#//apple_ref/doc/uid/TP30001066-CH220-BAJFDIDA)

2006-10-03Made minor technical improvements.

 Added cross references to the reference documentation for the constants listed in  [Table 2-1](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-CJBEAGHH).

 Removed information on using a `CGGLContextRef` object because the use of a graphics context for OpenGL rendering is not reliable and is not recommended.

 Added thread safety information to [Creating a PostScript Converter Object](dq_ps_convert/dq_ps_convert.html#//apple_ref/doc/uid/TP30001066-CH215-SW1).

2006-07-24Made minor technical improvements.

 Changed [Listing 2-6](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-BCICJBGG) so that is correctly frees the bitmap data.

 Added cross references to [Creating an Image From Part of a Larger Image](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-SW1) and [Creating an Image from a Bitmap Graphics Context](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-SW2) that link to examples of creating graphics contexts.

2006-06-28Made minor changes to clarify a few concepts.

 Revised [Figure 12-2](dq_layers/dq_layers.html#//apple_ref/doc/uid/TP30001066-CH219-SW2) and the text that describes it.

 Revised [Figure 1-2](dq_overview/dq_overview.html#//apple_ref/doc/uid/TP30001066-CH202-BBCJFFEJ) and the text that describes it.

 Revised information in “Python Bindings for Quartz 2D”.

 Provided hyperlinks to the functions and methods discussed in [Data Management in Quartz 2D](dq_data_mgr/dq_data_mgr.html#//apple_ref/doc/uid/TP30001066-CH216-TPXREF101).

 Corrected a typographical error in [Listing 2-2](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-CJBGGCCC).

2006-02-07Corrected typographical error.

2006-01-10Made minor typographical and technical corrections.

2005-11-09Corrected several technical, typographical, and formatting errors.

 Made changes to code in [Listing 12-1](dq_layers/dq_layers.html#//apple_ref/doc/uid/TP30001066-CH219-SW1).

 Revised introductory paragraphs in [Transforms](dq_affine/dq_affine.html#//apple_ref/doc/uid/TP30001066-CH204-SW1).

 Revised several sentences in How Quartz 2D Draws Text.

2005-07-07Corrected typos and added clarification about Quartz OpenGL graphics context.

2005-06-04Fixed typos and added a Python script name.

2005-04-29Updated for OS X v10.4.

 Changed the title from *Drawing With Quartz 2D* to make it more consistent with the titles of similar documentation.

 Revised the Introduction to reflect the new content.

 Simplified the code in [Figure 3-1](dq_paths/dq_paths.html#//apple_ref/doc/uid/TP30001066-CH211-CJBGFABH).

 Revised the introductions for [Color and Color Spaces](dq_color/dq_color.html#//apple_ref/doc/uid/TP30001066-CH205-TPXREF101), [Transforms](dq_affine/dq_affine.html#//apple_ref/doc/uid/TP30001066-CH204-TPXREF101), [Bitmap Images and Image Masks](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-TPXREF101), and [PDF Document Parsing](dq_pdf_scan/dq_pdf_scan.html#//apple_ref/doc/uid/TP30001066-CH220-TPXREF101).

 Made changes to code in [Code that uses layers to draw a flag](dq_layers/dq_layers.html#//apple_ref/doc/uid/TP30001066-CH219-BAJCIBEE) so that more appropriately-sized layers are used; substituted the function `CGContextDrawLayerAtPoint` for `CGContextDrawLayerInRect`.

 Revised the section [Setting Blend Modes](dq_paths/dq_paths.html#//apple_ref/doc/uid/TP30001066-CH211-BCIGICEF); added figures that show actual output produced using blend modes.

 Revised the section [Using Blend Modes with Images](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-CJBIJEFG) and replaced the figures with better examples of drawing an image using different blend modes.

 Added information about Core Image and Core Video in the opening paragraphs of [Overview of Quartz 2D](dq_overview/dq_overview.html#//apple_ref/doc/uid/TP30001066-CH202-TPXREF101).

 Introduced the notion of CGLayer objects in the section [Drawing Destinations: The Graphics Context](dq_overview/dq_overview.html#//apple_ref/doc/uid/TP30001066-CH202-TPXREF131).

 Added the new Tiger opaque objects to [Quartz 2D Opaque Data Types](dq_overview/dq_overview.html#//apple_ref/doc/uid/TP30001066-CH202-BBCGCGBA).

 Added blend mode to [Graphics States](dq_overview/dq_overview.html#//apple_ref/doc/uid/TP30001066-CH202-TPXREF132). Added information about using blend modes to [Paths](dq_paths/dq_paths.html#//apple_ref/doc/uid/TP30001066-CH211-TPXREF101) and [Bitmap Images and Image Masks](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-TPXREF101).

 Revised [Graphics Contexts](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-TPXREF101) to show how to use HIView. Also added new figures to many sections and provided information on HIView coordinates as compared to Quartz coordinates.

 Added [Table 2-1](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-CJBEAGHH) to show the supported color spaces and pixel formats.

 Replaced [Figure 2-4](dq_context/dq_context.html#//apple_ref/doc/uid/TP30001066-CH203-BBCFJDBF) to show an enlargement of aliased and antialiasing drawing and text.

 Added [Ellipses](dq_paths/dq_paths.html#//apple_ref/doc/uid/TP30001066-CH211-BCIHJHEA) and revised discussions on [Painting a Path](dq_paths/dq_paths.html#//apple_ref/doc/uid/TP30001066-CH211-CJBEIHHJ) and [Clipping to a Path](dq_paths/dq_paths.html#//apple_ref/doc/uid/TP30001066-CH211-TPXREF126) to reflect new Tiger content.

 Changed “clipping region” to “clipping area” throughout the entire book.

 Revised information on [Creating Color Spaces](dq_color/dq_color.html#//apple_ref/doc/uid/TP30001066-CH205-TPXREF128) to reflect Tiger content.

 Added [Evaluating Affine Transforms](dq_affine/dq_affine.html#//apple_ref/doc/uid/TP30001066-CH204-CJBJGCGI) and [Getting the User to Device Space Transform](dq_affine/dq_affine.html#//apple_ref/doc/uid/TP30001066-CH204-CJBIGDFB).

 Revised the chapter formerly titled *Data Providers and Data Consumers* to contain information on image sources and image destinations, and how to move data between Quartz 2D and Core Image. Retitled the chapter [Data Management in Quartz 2D](dq_data_mgr/dq_data_mgr.html#//apple_ref/doc/uid/TP30001066-CH216-TPXREF101) to reflect the revised content.

 Renamed the Bitmap Image chapter to [Bitmap Images and Image Masks](dq_images/dq_images.html#//apple_ref/doc/uid/TP30001066-CH212-TPXREF101) and substantially revised the content to reflect information about image sources, the new image creation functions, image masking function, and using blend modes to composite images. 

 Added the chapter [Core Graphics Layer Drawing](dq_layers/dq_layers.html#//apple_ref/doc/uid/TP30001066-CH219-TPXREF101).

 Added the chapter [PDF Document Parsing](dq_pdf_scan/dq_pdf_scan.html#//apple_ref/doc/uid/TP30001066-CH220-TPXREF101), which contains some material from the old PDF Document chapter along with new material on scanners and content streams.

 Added Copying Font Variations and PostScript Fonts to the Text chapter.

2004-06-28Revised for Mac OS X v10.3.
